# /public/photos — Photo Drop Folder

Drop all client and coach photos here. Filenames must match exactly as listed below.

| Filename | Description |
|---|---|
| `jerome-before-after.jpg` | Jerome's before/after — front relaxed comparison (used in Results section) |
| `ben-before-after.jpg` | Ben's before/after (add when ready) |
| `phil-before-after.jpg` | Phil's before/after (add when ready) |
| `tenhang-coach-portrait.jpg` | Coach portrait (not used yet — reserve for future) |
| `tenhang-japan-travel.jpg` | Japan travel lifestyle shot of Tenhang — NOT a physique photo. Used in [04] — ABOUT THE COACH section (left column photo placeholder) |
| `gallery-1.jpg` – `gallery-8.jpg` | Gallery section (3 travel · 2 food · 2 basketball · 1 gym) |

## Format

- JPG preferred, PNG accepted
- Minimum 800px on shortest side
- Portrait orientation preferred for before/after cards

## How it works

Each photo slot has a branded placeholder visible until the real file is dropped in.
Once a file with the correct name is placed here, it automatically replaces the placeholder on the next page load — no code changes needed.
